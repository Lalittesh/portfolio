# Lalittesh Kumar – Developer Portfolio

A premium, modern developer portfolio built with **React + Vite**.

## Features

- ⚡ Animated mesh gradient background
- 🎨 Dark navy + electric teal design
- ✍️ Typing animation in hero section
- 🔍 Scroll reveal animations
- 📊 Animated skill progress bars
- 🃏 Interactive project cards with hover effects
- 📱 Fully responsive + hamburger menu
- 💾 Resume download button

---

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Add your resume

Place your resume PDF inside the `public/` folder and name it:

```
public/resume.pdf
```

### 3. Run locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## Build for Production

```bash
npm run build
```

The output will be in the `dist/` folder. Deploy it to **Vercel**, **Netlify**, or **GitHub Pages**.

---

## Deploy to Vercel (Recommended)

1. Push this project to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and import the repo
3. Vercel auto-detects Vite — click **Deploy**
4. Done! Your portfolio is live 🚀

---

## Customization

All your personal data is in `src/Portfolio.jsx`:

- `SKILLS` array – edit skill names and percentages
- `PROJECTS` array – edit project titles, descriptions, tech stack, and GitHub links
- Hero section – update bio text and button links
- Contact section – all info is inline in the JSX

---

## Tech Stack

- React 18
- Vite 5
- CSS (custom, no framework needed)
- Google Fonts: Syne + DM Sans + JetBrains Mono
